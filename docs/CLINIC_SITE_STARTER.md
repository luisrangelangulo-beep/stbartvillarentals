# Christian Clinic Starter Map

This child theme now includes the minimum structure to begin development.

## Included now

- functions.php (theme setup, menus, assets, build marker)
- style.css (theme registration header)
- assets/css/site.css (site design system + layout styles)
- header.php / footer.php (global shell)
- front-page.php (starter homepage sections)
- page.php (default page rendering)

## Recommended next templates

- page-about.php
- page-services.php
- page-contact.php

- single-post.php (if blog/news)
- archive.php (if blog/news)

## WordPress setup checklist

1. Create menus and assign locations:
- Primary Menu
- Footer Menu

2. Set homepage:
- Settings > Reading > Your homepage displays > A static page
- Choose your Home page

3. Create starter pages:
- Home
- About
- Services
- Contact
- Book Appointment

4. Optional plugin stack:
- SEO plugin (Rank Math or Yoast)
- Forms plugin (WPForms / Fluent Forms)
- Security + backups
- Caching plugin
